## Design & UI tooling

For any work involving a UI decision or a new feature/idea, consult these tools before implementing:

- **21st MCP** (`mcp__21st__search`, `mcp__21st__get_component`, etc.) — search for component patterns and inspiration before building or redesigning UI.
- **ui-ux-pro-max** skill — query it for design-system guidance (styles, color palettes, typography, spacing, UX guidelines) before making UI/UX decisions.

Use both for:
- New pages, components, or UI redesigns
- Any visual/UX decision (layout, color, typography, spacing, interaction/motion patterns)
- Brainstorming or scoping a new feature idea that has a user-facing UI surface

Skip both for pure backend/logic/config work with no visual impact.
